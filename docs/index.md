# moralhazard

Moral hazard problem solver implementing Azevedo and Woff (2025).

## Installation

```bash
uv add git+https://github.com/eduardomazevedo/moralhazard.git
```

## API Reference

- [MoralHazardProblem](reference/problem.md) - Main problem class
- [Configuration](reference/config_maker.md) - Utility and distribution factories
- [Result Types](reference/types.md) - Solver output containers
